# OneClickVirt - containerd

Containerd Container Runtime Deployment

## 快速开始

```bash
# 查看帮助
bash containerdinstall.sh --help

# 安装
bash containerdinstall.sh
```

## 功能特性

- 📦 一键安装和部署
- 🚀 完整的生命周期管理
- 📊 监控和告警
- 🔒 安全隔离
- ⚡ 高性能

## 项目内容

本仓库包含以下脚本:

```
/containerdinstall.sh              - 安装脚本
/scripts/                          - 管理和配置脚本
/test.sh                           - 测试脚本
```

## 使用示例

```bash
# 查看所有可用命令
ls -la scripts/

# 执行特定脚本
bash scripts/[script-name].sh
```
